# 🛒 Grocery App (Redux)

A simple **Grocery Shopping Cart Application** built with **React and Redux**.
This project was created for **learning and practicing Redux state management**.

---

## ✨ Features

✅ Add grocery products
✅ Add products to cart
✅ Increase product quantity
✅ Decrease product quantity
✅ Remove products from cart
✅ Automatic total price calculation
✅ State management using Redux

---

## ⚙️ How It Works

1️⃣ Products are displayed in the product list
2️⃣ Users can add products to the cart
3️⃣ Redux manages the application state
4️⃣ Users can increase or decrease product quantity
5️⃣ If quantity becomes **0**, the product is automatically removed
